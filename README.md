# pre-debloated-windows

windows 11 pro images that come with Explorer Patcher

use 'irm https://get.activated.win | iex' to activate

build #1.1 is out btw

creds to ntdev for tiny11builder 

https://github.com/ntdevlabs/tiny11builder





how to use 
1. download the iso
2. burn the iso to a usb with ventoy, rufus or balena etcher
3. boot the usb
4. install like normal windows
5. do shift+f10 then type 'taskmgr' then force close windows update during oobe (not necessary)
6. when reached desktop do win+x then click 'terminal (admin)' then type 'irm https://get.activated.win | iex' and follow the steps (also not necessary)
7. do win then 'properties (explorer patcher)' to adjust the taskbar to your needs
