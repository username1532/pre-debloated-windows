# pre-debloated-windows
windows 11 pro images that come with explorer patcher
use 'irm https://get.activated.win | iex' to activate
build #1 is out btw
creds to ntdev for tiny11builder 
https://github.com/ntdevlabs/tiny11builder





how to use 
1. install like normal windows
2. do shift+f10 then type 'taskmgr' then force close windows update during oobe (not necessary)
3. when reached desktop do win+x then click 'terminal (admin)' then type 'irm https://get.activated.win | iex' and follow the steps (also not necessary)
4. do win then 'properties (explorer patcher)' to adjust the taskbar to your needs
